欢迎来到我的个人博客。

这个站点使用 **Next.js** 构建，文章以 Markdown 格式编写，存放在 `content/posts/` 目录下。简约、快速、易于部署——正是我想要的。

## 为什么写博客

> 写作是最好的思考方式。

- 沉淀知识，避免重复踩坑
- 分享经验，帮助他人
- 记录成长，回望来时路

## 技术栈

| 技术 | 用途 |
|------|------|
| Next.js | 框架 |
| Tailwind CSS | 样式 |
| Markdown | 内容 |
| Vercel / 海外 VPS | 部署 |

## 下一步

本地开发完成后，可以部署到 Vercel、Cloudflare Pages 或任何支持 Node.js 的海外服务器。

```bash
npm run build
npm run start
```
