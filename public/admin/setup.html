博客后台配置说明

本站使用 Cloudflare + 自建 OAuth，不会走 Netlify 登录。

1. 创建 GitHub OAuth 应用

  1. 打开 GitHub → Settings → Developer settings → OAuth Apps → New
  2. Homepage URL：
  3. Authorization callback URL：
  4. 记下 Client ID 和 Client Secret

2. Cloudflare Pages 环境变量

项目 Settings → Environment variables → Production，添加：

GITHUB_CLIENT_ID = 你的 Client ID
GITHUB_CLIENT_SECRET = 你的 Client Secret
ALLOWED_ORIGIN = quna.fun,www.quna.fun

若只用单一域名访问，可只填一个，例如

3. 重新部署后登录

返回后台，应跳转到 /auth 再跳转 GitHub，而不是 api.netlify.com。
